不是另一個 AI reviewer

不能 deterministic，就不阻擋 CI。

Evidrift 會做

  • 記錄某個 code location 為何依賴一項 static contract；
  • 透過 CLI、MCP 或 GitHub Actions 重新驗證同一份 evidence；
  • 在 deterministic contract mismatch 或 evidence-integrity failure 時失敗；
  • source 改變但 selected contract 仍相同時只提出 warning。

Evidrift 不會做

  • 證明 runtime correctness 或 free-text semantic truth；
  • 抓取 arbitrary URL 或解析 remote OpenAPI reference；
  • 執行 package code 或 Receipt 提供的 command；
  • 把 LLM judge 當成 CI gate。
